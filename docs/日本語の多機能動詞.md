## 第1章：複合動詞とは何か

### 1.1 基本構造

{複合動詞(ふくごうどうし)}は**{前項動詞(ぜんこうどうし)}（V1）の{連用形(れんようけい)}＋{後項動詞(こうこうどうし)}（V2）**で{構成(こうせい)}される：

```
言う（いう）→ 言い（いい）+ 始める → 言い始める（いいはじめる）= to start to say
食べる      → 食べ + 始める → 食べ始める = to start eating
走る        → 走り + 出す   → 走り出す  = to start running suddenly
```

V1が核(かく)となる{動作(どうさ)}を{提供(ていきょう)}し、V2がその{動作(どうさ)}に「{方向性(ほうこうせい)}」「アスペクト（{時間的側面(じかんてきそくめん)}）」「{程度(ていど)}」などのニュアンスを{追加(ついか)}する。{複合動詞全体(ふくごうどうしぜんたい)}はV2の{活用(かつよう)}パターン（{一段(いちだん)}・{五段(ごだん)}）に従(じゅう)って{活用(かつよう)}する。

### 1.2 二つのタイプ

{複合動詞(ふくごうどうし)}は大(だい)きく二(に)つに{分類(ぶんるい)}される：

| タイプ | 特徴 | 例 |
|--------|------|-----|
| **統語的複合動詞** | V2が文法機能語化。どんなV1とも自由に結合可能。前項に「〜する」、受身、使役も入る | 〜始める、〜過ぎる、〜続ける、〜直す、〜合う |
| **語彙的複合動詞** | V1+V2の組み合わせが限定。慣用的に固定された意味 | 思い出す、落ち着く、見積もる |

{統語的複合動詞(とうごてきふくごうどうし)}は{英語(えいご)}の phrasal verbs に最(さい)も近(きん)く、V2を「{不変化詞(ふへんかし)}」的(てき)に使(し)える。{一方(いっぽう)}、{語彙的複合動詞(ごいてきふくごうどうし)}は{個別暗記(こべつあんき)}が{必要(ひつよう)}だが、{一度覚(いちどかく)}えれば{表現力(ひょうげんりょく)}が{格段(かくだん)}に広(こう)がる。

## 第2章：主要な多機能後項動詞（V2）カタログ

{英語(えいご)}の "give up / give in / give out" では "up" "in" "out" が{意味(いみ)}を変(へん)える。{日本語(にほんご)}では{以下(いか)}の{後項動詞(こうこうどうし)}がそれに{相当(そうとう)}する。

### 2.1 開始を表す — 「始める」vs「出す」

| 後項 | 読み | ニュアンス | 例 |

| 〜始める | はじめる | 計画的・漸進的な開始。意識的な行動や自然な過程 | 食べ始める、勉強し始める、降り始める |
| 〜出す | だす | 突然の・自発的な開始。制御不能な burst | 泣き出す（burst into tears）、走り出す（break into a run）、笑い出す（burst out laughing） |

### 2.2 継続・完了を表す

| 後項 | 読み | コア意味 | 英語的対応 | 例 |
|------|------|---------|-----------|-----|
| 〜続ける | つづける | 継続 | keep doing | 働き続ける、考え続ける、増え続ける |
| 〜終わる | おわる | 自然な完了 | finish doing | 食べ終わる、読み終わる、書き終わる |

| 〜尽くす | つくす | 残さずやり尽くす | exhaust / do to the fullest | 食べ尽くす（eat everything）、語り尽くす（talk thoroughly） |

{複合動詞(ふくごうどうし)}だけでなく、{日本語(にほんご)}の{基本動詞自体(きほんどうしじたい)}が{多様(たよう)}な{意味(いみ)}を持(じ)つものも多(た)い。{{{{国立国語研究所(こくりつこくごけんきゅうしょ)}(こくりつこくごけんきゅうしょ)}(こくりつこくごけんきゅうしょ)}(こくりつこくごけんきゅうしょ)}の「{基本動詞(きほんどうし)}ハンドブック」では**190語(ご)の{多義動詞(たぎどうし)}**を{収録(しゅうろく)}している。

### 4.1 とる（取る・撮る・執る・採る）

「手(て)でつかむ」がプロトタイプ（{基本義(きほんぎ)}）：

1. **{物理的(ぶつりてき)}に取(しゅ)る**：手(て)に取(しゅ)る、席(せき)を取(しゅ)る、本(ほん)を取(しゅ)る
2. **{取得(しゅとく)}する**：休(きゅう)みを取(しゅ)る、{資格(しかく)}を取(しゅ)る、{点数(てんすう)}を取(しゅ)る
3. **{記録(きろく)}する**：{写真(しゃしん)}を撮(さつ)る、メモを取(しゅ)る
4. **{除去(じょきょ)}する**：汚(お)れを取(しゅ)る、{塩分(えんぶん)}を取(しゅ)る
5. **{選択(せんたく)}する**：Aを取(しゅ)る（選(せん)ぶ）
6. **{摂取(せっしゅ)}する**：{食事(しょくじ)}を取(しゅ)る、{栄養(えいよう)}を取(しゅ)る
7. **{理解(りかい)}する**：{意味(いみ)}を取(しゅ)る（interpret）
8. **{時間(じかん)}を要(よう)する**：{手間取(てましゅ)}る（take time）

### 4.2 かける

「何(なに)かを何(なに)かにのせる」がコア：

1. **{物理的(ぶつりてき)}にかける**：{眼鏡(めがね)}をかける、{布団(ふとん)}をかける
2. **{通信(つうしん)}・働(どう)きかけ**：{電話(でんわ)}をかける、声(こえ)をかける、話(はなし)しかける
3. **{設置(せっち)}・{構築(こうちく)}**：橋(はし)をかける、罠(わな)をかける
4. **{経過(けいか)}**：{時間(じかん)}がかかる
5. **{負荷(ふか)}**：{負担(ふたん)}をかける、{迷惑(めいわく)}をかける、{圧力(あつりょく)}をかける
6. **{複合動詞(ふくごうどうし)}のV2**：浴(よく)びせかける、襲(しゅう)いかかる、呼(こ)びかける

### 4.3 はる（張る）

「{面状(めんじょう)}に広(こう)げる」がコア：

1. **貼(ちょう)る**：ポスターを張(ちょう)る、のりで張(ちょう)る
2. **張(ちょう)り巡(じゅん)らす**：ロープを張(ちょう)る、網(あみ)を張(ちょう)る
3. **伸(しん)ばす**：胸(むね)を張(ちょう)る、肩(かた)を張(ちょう)る
4. **（{値段(ねだん)}が）高(たか)い**：値(あたい)が張(ちょう)る
5. **固(こ)くなる**：{道路(どうろ)}が張(ちょう)る（凍(とう)る）
6. **{複合動詞(ふくごうどうし)}**：張(ちょう)り切(せつ)る、張(ちょう)り合(ごう)う、張(ちょう)り込(こみ)む

### 4.4 ぬく（抜く）

「中(なか)から外(そと)へ引(ひき)き出(しゅつ)す」がコア：

1. **{物理的(ぶつりてき)}に抜(ばつ)く**：歯(は)を抜(ばつ)く、栓(せん)を抜(ばつ)く、刀(かたな)を抜(ばつ)く
2. **選(せん)び出(しゅつ)す**：{選抜(せんばつ)}する、データを抜(ばつ)く
3. **追(つい)い越(えつ)す**：人(にん)を抜(ばつ)く（overtake）
4. **{除去(じょきょ)}する**：疲(ひ)れを抜(ばつ)く、アクを抜(ばつ)く
5. **{複合動詞(ふくごうどうし)} V2**：戦(せん)い抜(ばつ)く（fight through）、考(こう)え抜(ばつ)く（think through）

このように、一(いち)つの{動詞(どうし)}がメタファーによって{意味(いみ)}を{拡張(かくちょう)}していく{様子(ようす)}は、{英語(えいご)}の "give" や "take" と全(ぜん)く同(どう)じ{現象(げんしょう)}である。
