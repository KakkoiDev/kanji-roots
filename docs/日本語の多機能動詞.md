## はじめに

英語の "give" は "give up"（諦める）、"give in"（屈服する）、"give out"（配る／尽きる）、"give away"（譲る／暴露する）のように、後ろに付く不変化詞（particle）によって意味が劇的に変化する。これは日本語学習者にとって馴染み深いパターンだが、実は日本語にも全く同じ原理で機能する「多機能動詞」の仕組みが存在する。

日本語の多機能動詞は主に**二つの形**で現れる：

1. **複合動詞（ふくごうどうし）** — 「動詞＋動詞」の組み合わせで、後ろの動詞が英語の不変化詞のような役割を果たす
2. **多義動詞（たぎどうし）** — 一つの動詞自体が核となる意味から放射状に意味を拡張する

本稿では、特に**複合動詞の後項動詞（V2）**に焦点を当て、英語の phrasal verbs との対比も交えながら、日本語の多機能動詞体系を整理する。

## 第1章：複合動詞とは何か

### 1.1 基本構造

複合動詞は**前項動詞（V1）の連用形＋後項動詞（V2）**で構成される：

```
言う（いう）→ 言い（いい）+ 始める → 言い始める（いいはじめる）= to start to say
食べる      → 食べ + 始める → 食べ始める = to start eating
走る        → 走り + 出す   → 走り出す  = to start running suddenly
```

V1が核となる動作を提供し、V2がその動作に「方向性」「アスペクト（時間的側面）」「程度」などのニュアンスを追加する。複合動詞全体はV2の活用パターン（一段・五段）に従って活用する。

### 1.2 二つのタイプ

複合動詞は大きく二つに分類される：

| タイプ | 特徴 | 例 |
|--------|------|-----|
| **統語的複合動詞** | V2が文法機能語化。どんなV1とも自由に結合可能。前項に「〜する」、受身、使役も入る | 〜始める、〜過ぎる、〜続ける、〜直す、〜合う |
| **語彙的複合動詞** | V1+V2の組み合わせが限定。慣用的に固定された意味 | 思い出す、落ち着く、見積もる |

統語的複合動詞は英語の phrasal verbs に最も近く、V2を「不変化詞」的に使える。一方、語彙的複合動詞は個別暗記が必要だが、一度覚えれば表現力が格段に広がる。

## 第2章：主要な多機能後項動詞（V2）カタログ

英語の "give up / give in / give out" では "up" "in" "out" が意味を変える。日本語では以下の後項動詞がそれに相当する。

### 2.1 開始を表す — 「始める」vs「出す」

英語の "start" に対応するが、ニュアンスの違いが極めて重要。

| 後項 | 読み | ニュアンス | 例 |
|------|------|-----------|-----|
| 〜始める | はじめる | 計画的・漸進的な開始。意識的な行動や自然な過程 | 食べ始める、勉強し始める、降り始める |
| 〜出す | だす | 突然の・自発的な開始。制御不能な burst | 泣き出す（burst into tears）、走り出す（break into a run）、笑い出す（burst out laughing） |

> **比較**：「雨が降り始めた」= 自然に降り出した（漸進的） vs 「突然、子どもが泣き出した」= 予期せぬ制御不能な開始

### 2.2 継続・完了を表す

| 後項 | 読み | コア意味 | 英語的対応 | 例 |
|------|------|---------|-----------|-----|
| 〜続ける | つづける | 継続 | keep doing | 働き続ける、考え続ける、増え続ける |
| 〜終わる | おわる | 自然な完了 | finish doing | 食べ終わる、読み終わる、書き終わる |
| 〜上げる | あげる | 仕上げ・完成 | complete / finish up | 書き上げる、縫い上げる、仕上げる |
| 〜切る | きる | 完全に〜する | do completely / use up | 使い切る（use up）、信じ切る（fully believe） |
| 〜尽くす | つくす | 残さずやり尽くす | exhaust / do to the fullest | 食べ尽くす（eat everything）、語り尽くす（talk thoroughly） |
| 〜抜く | ぬく | 最後までやり遂げる | see through / endure | 生き抜く（survive）、耐え抜く（endure through） |
| 〜通す | とおす | 貫通・貫徹 | carry through / persist | やり通す（see through）、読み通す（read through） |
| 〜果てる | はてる | 完全に（状態の極限） | completely (state) | 疲れ果てる（exhausted）、困り果てる（at a loss） |

特に **「〜上げる」** は「上に上げる」という物理的意味（見上げる）と「完成させる」というアスペクト的意味（書き上げる）の両方を持ち、多機能の典型例である。

### 2.3 相互・繰り返しを表す

| 後項 | 読み | コア意味 | 例 |
|------|------|---------|-----|
| 〜合う | あう | 相互に〜する | 話し合う（discuss together）、助け合う（help each other）、愛し合う（love each other） |
| 〜返す | かえす | やり返す・繰り返す | 言い返す（talk back）、繰り返す（repeat）、思い返す（reflect） |
| 〜直す | なおす | やり直す（修正） | 書き直す（rewrite）、やり直す（redo）、考え直す（reconsider） |

**〜合う** は特に生産的で、話し合い（noun: discussion）のように名詞化も可能。英語の "each other" の代わりに一語で表現できる強力なパターン。

### 2.4 空間・移動を表す（多義的王者：〜込む）

**〜込む（こむ）** は最も多機能な後項動詞で、以下の5つの異なる意味を持つ：

| 意味カテゴリ | 例 | 説明 |
|-------------|-----|------|
| ① 物理的な中への移動 | 飛び込む（jump in）、押し込む（push into） | 文字通り「中に入る」 |
| ② 心理的な没入 | 考え込む（be lost in thought）、思い込む（be convinced） | 深く考える・思い込む |
| ③ 徹底的な動作 | 飲み込む（swallow / grasp）、話し込む（get deep in conversation） | 十分に行う |
| ④ 書き込み・打ち込み | 書き込む（fill in）、打ち込む（type in / devote to） | 容器・媒体に入れる |
| ⑤ 詰め込み・割り込み | 詰め込む（cram）、割り込む（cut in） | 無理やり入れる |

この5つの意味はすべて「外から内へ」というコアイメージから派生しており、認知言語学的な意味拡張の好例となっている。

その他の空間的後項動詞：

| 後項 | 読み | 意味 | 例 |
|------|------|------|-----|
| 〜上がる（自） | あがる | 上へ／完成 | 立ち上がる（stand up）、でき上がる（be finished） |
| 〜上げる（他） | あげる | 上へ上げる／仕上げ | 見上げる、書き上げる |
| 〜渡る | わたる | 全体に及ぶ | 晴れ渡る（clear all over）、響き渡る（resound） |
| 〜散らす | ちらす | あちこちに | 書き散らす、まき散らす、怒鳴り散らす |

### 2.5 失敗・否定を表す

英語の phrasal verbs には "miss" や "fail" に相当する不変化詞はないが、日本語には専用の後項動詞が存在する。

| 後項 | 読み | 意味 | 例 |
|------|------|------|-----|
| 〜損なう | そこなう | 〜しそこなう | 食べ損なう、見損なう、聞き損なう |
| 〜忘れる | わすれる | 〜し忘れる | 言い忘れる（forget to say）、置き忘れる（leave behind） |
| 〜そびれる | そびれる | 機会を逃す | 食べそびれる、聞きそびれる |
| 〜かねる | かねる | 〜できない（控えめ） | 引き受けかねる（cannot take on）、決めかねる（cannot decide） |
| 〜誤る | あやまる | 〜し間違える | 書き誤る、判断し誤る |

### 2.6 習慣・能力を表す

| 後項 | 読み | 意味 | 例 |
|------|------|------|-----|
| 〜慣れる | なれる | 慣れる | 使い慣れる、食べ慣れる、聞き慣れる |
| 〜飽きる | あきる | 飽きる | 食べ飽きる、聞き飽きる |
| 〜こなす | こなす | 使いこなす | 乗りこなす（manage to drive）、使いこなす（use skillfully） |

## 第3章：多機能前項動詞（V1）

英語の "give" が多様な不変化詞と結合するように、日本語の一部の動詞（V1）も多様なV2と結合して豊かな意味を生み出す。

### 取り〜（とり）← 取る（とる）= to take

「取る」は「手に取る」が基本義だが、複合動詞の前項として極めて生産的：

| 複合動詞 | 意味 | 構成 |
|---------|------|------|
| 取り上げる | pick up / take up / confiscate | 取る + 上げる |
| 取り入れる | adopt / take in | 取る + 入れる |
| 取り消す | cancel / take back | 取る + 消す |
| 取り組む | tackle / engage in | 取る + 組む |
| 取り締まる | control / regulate | 取る + 締まる |
| 取り戻す | take back / recover | 取る + 戻す |
| 取り調べる | investigate | 取る + 調べる |
| 取り替える | exchange / replace | 取る + 替える |

### 引き〜（ひき）← 引く（ひく）= to pull

「引く」の「手前に引き寄せる」がコア。そこから比喰的拡張：

| 複合動詞 | 意味 |
|---------|------|
| 引き受ける | take on (responsibility) |
| 引き出す | pull out / withdraw |
| 引き寄せる | attract / pull toward |
| 引き続く | continue / follow |
| 引き返す | turn back |
| 引き止める | stop / detain |
| 引き渡す | hand over / extradite |

### 見〜（み）← 見る（みる）= to see

視覚から比喰へ拡張：

| 複合動詞 | 意味 |
|---------|------|
| 見上げる | look up |
| 見下ろす | look down |
| 見送る | see off / forgo |
| 見積もる | estimate |
| 見守る | watch over |
| 見逃す | miss / overlook |
| 見抜く | see through / detect |
| 見渡す | survey |

### 受け〜（うけ）← 受ける（うける）= to receive

| 複合動詞 | 意味 |
|---------|------|
| 受け入れる | accept (person / idea) |
| 受け取る | receive / accept (package) |
| 受け付ける | tolerate / accept (applications) |
| 受け持つ | be in charge of |
| 受け継ぐ | inherit / succeed |
| 受け止める | catch / react to |

### その他の多機能V1

| 前項 | 読み | 元の意味 | 結合例 |
|------|------|---------|--------|
| 立ち〜 | たち | 立つ | 立ち上がる、立ち寄る、立ち向かう、立ち直る、立ち入る |
| 押し〜 | おし | 押す | 押し切る、押し込む、押し通す、押し寄せる、押し付ける |
| 打ち〜 | うち | 打つ | 打ち合わせる、打ち切る、打ち消す、打ち上げる、打ち込む |
| 追い〜 | おい | 追う | 追いかける、追い越す、追い込む、追い出す、追い払う |
| 飛び〜 | とび | 飛ぶ | 飛び込む、飛び出す、飛び立つ、飛び回る、飛び付く |
| 吹き〜 | ふき | 吹く | 吹き出す、吹き飛ばす、吹き込む、吹き荒れる |
| 持ち〜 | もち | 持つ | 持ち上げる、持ち込む、持ち出す、持ち帰る、持ち歩く |
| 繰り〜 | くり | 繰る | 繰り返す、繰り上げる、繰り下げる、繰り出す |
| 乗り〜 | のり | 乗る | 乗り換える、乗り込む、乗り越える、乗り出す |
| 踏み〜 | ふみ | 踏む | 踏み切る、踏み込む、踏み出す、踏み外す |

## 第4章：一語で多機能な動詞（多義動詞）

複合動詞だけでなく、日本語の基本動詞自体が多様な意味を持つものも多い。国立国語研究所の「基本動詞ハンドブック」では**190語の多義動詞**を収録している。

### 4.1 とる（取る・撮る・執る・採る）

「手でつかむ」がプロトタイプ（基本義）：

1. **物理的に取る**：手に取る、席を取る、本を取る
2. **取得する**：休みを取る、資格を取る、点数を取る
3. **記録する**：写真を撮る、メモを取る
4. **除去する**：汚れを取る、塩分を取る
5. **選択する**：Aを取る（選ぶ）
6. **摂取する**：食事を取る、栄養を取る
7. **理解する**：意味を取る（interpret）
8. **時間を要する**：手間取る（take time）

### 4.2 かける

「何かを何かにのせる」がコア：

1. **物理的にかける**：眼鏡をかける、布団をかける
2. **通信・働きかけ**：電話をかける、声をかける、話しかける
3. **設置・構築**：橋をかける、罠をかける
4. **経過**：時間がかかる
5. **負荷**：負担をかける、迷惑をかける、圧力をかける
6. **複合動詞のV2**：浴びせかける、襲いかかる、呼びかける

### 4.3 はる（張る）

「面状に広げる」がコア：

1. **貼る**：ポスターを張る、のりで張る
2. **張り巡らす**：ロープを張る、網を張る
3. **伸ばす**：胸を張る、肩を張る
4. **（値段が）高い**：値が張る
5. **固くなる**：道路が張る（凍る）
6. **複合動詞**：張り切る、張り合う、張り込む

### 4.4 ぬく（抜く）

「中から外へ引き出す」がコア：

1. **物理的に抜く**：歯を抜く、栓を抜く、刀を抜く
2. **選び出す**：選抜する、データを抜く
3. **追い越す**：人を抜く（overtake）
4. **除去する**：疲れを抜く、アクを抜く
5. **複合動詞 V2**：戦い抜く（fight through）、考え抜く（think through）

このように、一つの動詞がメタファーによって意味を拡張していく様子は、英語の "give" や "take" と全く同じ現象である。

## 第5章：国立国語研究所「複合動詞レキシコン」の分類

学術的に最も信頼できるリソースとして、国立国語研究所の「複合動詞レキシコン」（2,700語以上収録）では、複合動詞を以下の4構造に分類している。

### VV型（動詞＋動詞）

V1とV2が両方とも本来の意味と格関係を保つ。「V1てV2」と言い換え可能。

| 複合動詞 | 言い換え | 意味 |
|---------|---------|------|
| 踏みつぶす | 踏んでつぶす | trample and crush |
| 押し開ける | 押して開ける | push open |
| 歩き疲れる | 歩いて疲れる | get tired from walking |
| 流れ着く | 流れて着く | drift to |

格関係は基本的にV2（後項）が決定する。

### Vs型（動詞＋補助動詞）

V2が補助動詞化し、文字通りの意味を失っている。「V1てV2」と言い換え不可能。代わりに副詞的にV1を修飾。格関係はV1が決定。

| 複合動詞 | 意味（言い換え） | 特徴 |
|---------|----------------|------|
| 晴れ渡る | すみずみまで晴れる | V2「渡る」は副詞的にV1を修飾 |
| 降りしきる | しきりに降る | V2「しきる」は単独では使えない |
| 咲き誇る | 誇らしげに咲く | V2「誇る」は比喰的に |
| 待ちわびる | わびるほど待つ | V2「わびる」は「待つ」としか結合しない |

Vs型のV2は以下の特徴を持つ：

- 語彙的アスペクトの意味を添加
- 単独で使われる時と意味が異なる
- 限られたV1としか結合しない
- 単独で使えないものがある
- 活用パラダイムが不完全なものがある

### pV型（接頭辞化した動詞＋動詞）

V1が接頭辞化し、V2の意味を強めるだけ。格関係はV2が決定。

| 複合動詞 | 元のV1 | 効果 |
|---------|--------|------|
| 差し迫る | 差す | 迫る（強調）|
| ぶっ飛ばす | 打つ | 飛ばす（強調・勢い）|
| 打ち重なる | 打つ | 重なる（強調）|
| ひっぱたく | 引く | 叩く（強調）|

### V型（一語化）

現代人の語感では2語の複合動詞というより、一語として固定化。

| 動詞 | 分解 | 意味 |
|------|------|------|
| 落ち着く | 落ちる + 着く | calm down / settle |
| 思い出す | 思う + 出す | remember |
| 折り入る | 折る + 入る | privately / confidentially（折り入って）|

## 結論：日本語の多機能動詞体系の全体像

日本語の多機能動詞を理解するには、以下の3層を押さえるとよい。

### 第1層：統語的複合動詞
後項（V2）を「不変化詞」のように使う。覚えるのは約30パターン。前項は自由。

- 英語の phrasal verbs に最も近い
- 〜始める、〜出す、〜続ける、〜終わる、〜過ぎる、〜直す、〜合う など
- 使いこなせれば N3 レベル

### 第2層：語彙的複合動詞
前項（V1）を「多機能ベース」として使う。前項約20種。

- 一つ一つの複合動詞を覚える必要があるが、V1のコア意味から推測可能
- 取り〜、引き〜、見〜、受け〜、立ち〜、押し〜 など
- N2〜N1 レベルで重要

### 第3層：多義動詞（一語）
一語でありながら複数の意味を持つ基本動詞。約190語。

- メタファーによる意味拡張を理解する
- とる、かける、はる、ぬく、ながす、もつ など
- 初級から上級まで継続的に学習
